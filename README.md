# Quick Cover

Quick Cover is a Playnite Generic Plugin for quickly setting a game's cover and background images from local files or URL.

## Features

- Add `Quick Cover` items to the game context menu in Playnite
- Set a game's cover image from a local file
- Set a game's background image from a local file
- Configure default cover and background images in plugin settings
- Apply configured default images to selected games
- Preview configured default images directly in the settings page

## How to use

Open `Add-ons -> Extension settings -> Generic -> Quick Cover` to configure default local file paths or remote image URLs for cover and background images.

After that, you can use Quick Cover from a game's context menu, or click the icon in the top-right area of the main Playnite window to quickly apply cover and background images.
